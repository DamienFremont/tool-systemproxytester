# tool-systemproxytester
Know the proxy you are behind and have no idea (because of secured network, no admin, etc)

Usage :
```
SystemProxyTester.main(null);
```
 Output (console) :
```
proxy type : HTTP
proxy hostname : proxy.toto.net
proxy port : 8080
```
